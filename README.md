# AE2 ME Interface Through

You must install [AE2](https://www.curseforge.com/minecraft/mc-mods/applied-energistics-2) before use it.

In my map I've a pillars of Molecular Assemblers and ME Interfaces, to access the MEI through the Molecular Assembles I created this simple mod.
So just place a Molecular Assemble besides an ME Interface and now you'll see a button to access it.

This mod uses [Cleanroon](https://github.com/CleanroomMC/Cleanroom).
